# Sharath P — Data Analyst Portfolio

Modern Data Analyst portfolio built with React, Vite, Tailwind CSS, Framer Motion, and Lucide icons.

## Tech Stack
- React
- Tailwind CSS
- Framer Motion
- Vite

## Projects Included
- HR Attrition Analytics & Prediction
- Hospital Readmission Dashboard

## Run Locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
